- **Local create/edit → Todoist push (동기)**: `add_task`/`complete_task`/`set_priority` 모두 source="todoist" task에 대해 즉시 Todoist API 호출

- **Tombstone soft-delete**: upstream DELETE → `status="deferred"` (immutable)
- **Mixed commit semantics** (명시적 boundary):
  - TodoService = flush only (router commits)
  - `todoist_sync` 서비스 함수 = service commits (schedule precedent)
  - `push_local_task` = flush only (chat turn commit과 통합)
- **retry_count max 5** → `sync_state="failed"`
- **MA-1 strict 3-key**: `{access_token, refresh_token: None, scope}` (Todoist refresh_token 미발급, None으로 schema 무변동)
